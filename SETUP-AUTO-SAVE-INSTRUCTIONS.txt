==========================================
   ThymeSheet Auto-Save Setup Guide
==========================================

Your auto-save scripts have been created! To enable automatic saving:

METHOD 1 - Automatic (Recommended):
------------------------------------
1. Right-click on "setup-auto-save.bat"
2. Select "Run as administrator"
3. Click "Yes" when Windows asks for permission
4. Done! Auto-save will run every 30 minutes

METHOD 2 - Manual Save Anytime:
--------------------------------
Simply double-click "auto-save.bat" whenever you want to save your work.
This commits your changes to git with a timestamp.

To Disable Auto-Save:
---------------------
Right-click "disable-auto-save.bat" and select "Run as administrator"

What Gets Saved:
----------------
- Jeremy-Time-Tracker.html
- main.js
- package.json

Auto-save only saves if there are actual changes, so it won't spam your git history with empty commits.
