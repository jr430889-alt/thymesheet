WHAT IS THYMESHEET?
================================================================================

ThymeSheet is a desktop time tracking application designed specifically for
consultants, freelancers, and professionals who need to track billable hours
across multiple clients and projects.

Key Features:
- One-click time tracking with start/stop timer
- Organize time by Client > Project > Task > Subtask
- Export time entries to CSV for billing and reporting
- Import project lists from CSV files
- Auto-pause detection for idle time
- Daily goal tracking and progress widgets
- Keyboard shortcuts for power users
- Completely offline - your data never leaves your computer
- Privacy-first design - no data collection

QUICK START
================================================================================

1. INSTALL
   - Run the ThymeSheet-Setup installer
   - Follow the on-screen instructions
   - Launch ThymeSheet

2. AUTOMATIC ACTIVATION
   - ThymeSheet will activate automatically on first launch
   - No manual license key entry needed
   - Your license is permanently tied to this computer

3. ADD PROJECTS
   Option A: Import from CSV (recommended for many projects)
     - Go to "Manage Projects" tab
     - Click "Import Projects from CSV"
     - Choose your CSV file with format:
       Client Name, Project/Admin Name, Task Code, Subtask Code

   Option B: Add manually (for a few projects)
     - Go to "Manage Projects" tab
     - Fill in the form at the bottom
     - Click "Add Project"

   Option C: Try demo data
     - Click "Load Demo Data" to see examples

4. START TRACKING
   - Click "Track Time" tab
   - Select a project from the dropdown
   - (Optional) Add a comment about what you're working on
   - Click "Start" or press Enter
   - The timer starts immediately!

5. STOP TRACKING
   - Click "Stop" or press Space when done
   - Your time entry is automatically saved
   - View it in "Recent Time Entries" below the timer

6. EXPORT FOR BILLING
   - Go to "Manage Projects" tab
   - Click "Export Time to CSV"
   - Choose date range (or export all)
   - Open the CSV file in Excel or Google Sheets
   - Use the data to create your invoices

WHERE IS MY DATA STORED?
================================================================================

Your time tracking data is stored locally on your computer in:

Your Documents folder > ThymeSheet folder

Typical location: C:\Users\[YourName]\Documents\ThymeSheet\

This means:
- Your data stays on your computer (never sent to cloud servers)
- Your data survives application updates and reinstalls
- You can backup this folder to protect your data
- You are responsible for backing up your data

IMPORTANT: Always backup your ThymeSheet folder regularly!

FIRST TIME USER TIPS
================================================================================

1. Start with Demo Data
   - Click "Load Demo Data" to see how projects are organized
   - Practice tracking time on demo projects
   - Export the demo data to see what the CSV looks like
   - Delete demo data when ready: Settings > Reset Application Data

2. Import Your Real Projects
   - Create a CSV file with your clients and projects
   - Format: Client Name, Project Name, Task Code, Subtask Code
   - Import via "Manage Projects" > "Import Projects from CSV"

3. Customize Settings
   - Go to Settings (gear icon or Ctrl+,)
   - Set your time rounding preference
   - Enable auto-pause detection
   - Set daily work hour goal
   - Enable floating timer if desired

4. Learn Keyboard Shortcuts
   - Ctrl+Shift+Space to start/stop timer quickly
   - Enter to start tracking selected project
   - Much faster than using the mouse!

5. Establish Backup Routine
   - Export to CSV weekly (or more often)
   - Backup Documents\ThymeSheet folder
   - Test your backup by trying to open exported CSV files
